  • Create issue

    Create issue

    Any user can create a request based on a feature they want to see or a bug theyâ€™ve encountered. They can also add a monetary reward to the request. After validation by the Ethereum blockchain, a smart contract is generated for the specific issue.

    discover our alpha release

  • Fund issue

    Fund issue

    Any developer can propose a solution to an issue. All proposed solutions get evaluated, and the most suitable solution is determined.

    discover our alpha release

  • Approve solution

    Approve solution

    After the issue is solved, all funds are transferred. The developer that came up with the best solution will be notified, and is then able to collect his reward.
